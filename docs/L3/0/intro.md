# Introduction : Écrire un discours sur l’amour

:::info[Exercice en binôme]
**Inventer un personnage et faites-lui écrire un discours philosophique sur l'amour qui sera ensuite lu en classe :**
  - Choisir un **genre de discours** : judiciaire, délibératif, démonstratif
  - Chercher une **question de départ** à propos de l'amour
  - **Écrire votre discours** en utilisant les 3 registres étudiés dans la leçon précédente : **logos**, **pathos**, **ethos**.
:::
